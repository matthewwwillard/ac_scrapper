# Animal Crossing Scrapper
Animal Crossing Scrapper written in Typescript/Node!

As the title says! A scrapper written in node! Data from https://animalcrossing.fandom.com/!

Collects the following;

1. Vilagers
2. Bugs (North & South)
3. Fish (North * South)
4. Fossils
5. DIYs

## How to Run

1. Clone
2. npm install
3. npm run scrape
4. data!
